# 熱聲實驗室網站

[thermoacoustictw.org](https://thermoacoustictw.org) 官方網站原始碼。

技術棧：Astro 5 + Tailwind CSS 4 + MDX + KaTeX，部署於 Cloudflare Pages。

## 開發

```bash
npm install
npm run dev      # http://localhost:4321
npm run build
npm run preview
```

需要 Node 22+（見 `.nvmrc`）。
