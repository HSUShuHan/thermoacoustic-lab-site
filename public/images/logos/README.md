# 就業單位 / 升學學校 標誌

放置官方標誌／校徽檔（建議 SVG，或去背 PNG，高度 ~40px 以上）。放好後，於
`src/data/alumni-outcomes.ts` 對應項目的 `logo` 欄填入路徑（如 `/images/logos/tsmc.svg`），
即會在成員頁與加入我們頁的「畢業去向」以「標誌＋名稱」晶片顯示；未填則只顯示名稱。

- `tokyo-electron.svg`（東京威力科創 / Tokyo Electron）
